# CorpNexus

CorpNexus is a comprehensive Django-based Company Management System featuring a modern, premium dark glassmorphism user interface.

## 🌟 Features

- **Employee Management:** Add, edit, view, and manage employee records with ease.
- **Department Organization:** Keep track of departments, their heads, and active projects.
- **Modern UI/UX:**
  - Stunning **Dark Glassmorphism** design scheme.
  - Interactive **live search** functionality for employees and departments.
  - Form validation with visual feedback and shake animations on error.
  - Dynamic **dashboard** with real-time stats and hover parallax effects.
  - Celebratory **confetti animations** upon successful actions.
- **Responsive Design:** Fully responsive layout with mobile-friendly navigation menus.

## 🛠️ Tech Stack

- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3 (Custom Glassmorphism Framework), Vanilla JavaScript
- **Database:** SQLite (default Django configuration)

## 🚀 Getting Started

### Prerequisites
Make sure you have Python installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bhuvangujjarbhuvan-ctrl/Employee_Management-_System.git
   cd Employee_Management-_System
   ```

2. Run database migrations:
   ```bash
   python manage.py migrate
   ```

3. Start the development server:
   ```bash
   python manage.py runserver
   ```

4. Open your browser and navigate to `http://127.0.0.1:8000/`.
